# Запуск приложения Blink

## 🚀 Быстрый старт

### Требования
- Node.js 16+ https://nodejs.org
- MongoDB https://www.mongodb.com/try/download/community
- Git

### Шаг 1: Установка MongoDB

Убедись, что MongoDB запущена:
```bash
mongod
```

Или используй MongoDB Atlas (облачный):
https://www.mongodb.com/cloud/atlas

### Шаг 2: Установка Backend

```bash
cd server
npm install
```

Отредактируй `server/.env` если нужно (по умолчанию уже заполнен).

Запуск:
```bash
npm run dev
```

Должно появиться:
```
✓ MongoDB подключён
✓ Redis подключён (или пропущено)
✓ Сервер Blink запущен на http://localhost:5000
✓ Socket.io готов на ws://localhost:5000
```

### Шаг 3: Установка Frontend

В новом терминале:

```bash
cd client
npm install
```

Запуск:
```bash
npm run dev
```

Браузер откроется на http://localhost:5173

## 🆘 Если что-то не работает

### Терминал не находит `npm`
```bash
node --version  # проверь что Node установлен
npm --version   # проверь что npm установлен
```

Переустанови Node.js если нужно.

### MongoDB connection timeout
```bash
# Запусти MongoDB самостоятельно
mongod
```

### Port 5000 уже занят
Отредактируй `server/.env`:
```env
PORT=5001
```

### Port 5173 уже занят
Закрой другие Vite приложения или отредактируй `client/vite.config.js`.

### Геолокация не работает
- Используй `localhost` (не IP адрес)
- Разреши доступ к геолокации когда браузер запросит
- Включи высокую точность (enableHighAccuracy)
